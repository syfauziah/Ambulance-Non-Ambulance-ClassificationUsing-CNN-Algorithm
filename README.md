# Ambulance-Non-Ambulance-ClassificationUsing-CNN-Algorithm
Proyek ini bertujuan untuk membangun model klasifikasi gambar menggunakan Convolutional Neural Network (CNN). Model yang dikembangkan akan mengklasifikasikan citra kendaraan mobil menjadi dua kategori utama: **Ambulance atau Non-Ambulance.**

# Dataset
Dataset yang digunakan untuk pelatihan dan evaluasi model berasal dari https://www.kaggle.com/datasets/mahmoudshaheen1134/ambulance-dataset

Total Data: 4749 gambar.

Jumlah Kelas: 2 kelas (ambulance dan non- ambulance).

Rincian Kelas:
ambulance: 2121 gambar.
non-ambulance: 2628 gambar.

# Pembagian Data: 
Dataset dibagi menjadi tiga set dengan rasio sebagai berikut:

Data Pelatihan (Training): 70%.

Data Validasi (Validation): 15%.

Data Pengujian (Test): 15%.
